# 장바구니 프로젝트

## 프로젝트 개요

- 장바구니 페이지는 담은 상품의 목록과 가격, 배송비를 확인하고 결제 모듈로 연결하는 기능을 제공함
- 참여한 팀원 : 나는야잠만보

## 개발 환경
React

## 설치 및 실행 방법

프로젝트를 로컬에서 실행하기 위한 설치 방법과 명령어

```
npm install
npm start
```

## 컴포넌트 구조
- BackButton : 뒤로가기 버튼 (장바구니 이전 페이지로 이동)
- PayButton : 결제하기 버튼 (결제 모듈로 연결)
- CartList : 담은 상품 목록과 수량에 따른 총 금액, 배송비를 표시하는 장바구니 페이지
    - Cart : 각 상품의 이름, 가격, 수량 표시시

## 테스트 방법

테스트 케이스 및 실행 방법

```
npm test
```
