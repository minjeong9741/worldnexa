# 슈킹 웹 애플리케이션 개발자 가이드라인
## 프로젝트 개요
### 목적
> 사용자에게 편리한 상품 탐색, 카드 등록, 결제 경험을 제공하는 반응형 쇼핑목 웹 애플리케이션을 개발한다.

### 범위
- 상품 목록부터 결제 완료까지의 전 과정 지원
- 상품 목록 페이지, 상품 상세 페이지, 장바구니, 결제 모듈, 결제 완료 페이지 포함
- 카드 등록 및 보안 처리
- 테스트 URL 제공

### 참여인원
나는야잠만보

### 제공 가치
- 사용자 : 상품 탐색부터 결제까지 직관적이고 편리한 흐름
- 개발자 : 컴포넌트 기반 설계로 유지보수 용이이

## 시스템 아키텍처
```
[사용자]
   |
   v
[React App]
 ├── 상품 목록 페이지 (ProductList)
 │   └── ProductCard
 │   └── Header (로고 + 장바구니 수 표시)
 ├── 장바구니 페이지 (CartList)
 │   └── Cart (상품 정보, 수량, 가격)
 │   └── BackButton / PayButton
 ├── 결제 모듈 (CardRegistration)
 │   ├── CardList
 │   ├── CardNumberInput
 │   ├── ExpiryDateInput
 │   ├── OwnerNameInput
 │   ├── CVCInput
 │   ├── CardPasswordInput
 └── 결제 완료 페이지
```
