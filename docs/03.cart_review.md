# 장바구니 페이지 리뷰

## 고객사 요구사항
1. 더 좋은 사용자 경험을 위해 여러 상품을 한번에 비교고 결정하기 좋은 환경이 필요합니다.
2. 상품의 가격, 수량에 따른 총 금액, 배송비를 한 눈에 볼 수 있게 표시해주세요.
3. 10만원 이상 구매시 배송비는 무료입니다.
4. 뒤로가기 버튼을 이용해 메인 상품 목록 페이지로 이동하게 해주세요.
5. 결제하기 버튼을 누르면 결제 모듈을 모달로 사용할 수 있도록 연동해주세요.
