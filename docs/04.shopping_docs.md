# 쇼핑몰 연동

## 프로젝트 개요
- 쇼핑몰 연동 프로젝트는 상품 목록, 상품 상세, 결제 완료 페이지를 유기적으로 연결하고, 데이터 일관성을 유지하면서 사용자에게 매끄러운 쇼핑경험을 제공하는 것을 목표로 함
- 참여한 팀원 : 나는야잠만보보

## 개발 환경
React, JavaScript, BootStrap5 

## 설치 및 실행 방법
프로젝트를 로컬에서 실행하기 위한 설치 방법과 명령어
```
npm install
npm start
```

```
shoppingmall

    │     ├── ProductCard.jsx
```

## 주요 함수와 기능
- ProductList : 전체 상품 목록을 표시하는 컴포넌트
- ProductDetail : 개별 상품의 상세 정보와 같은 브랜드의 관련 상품 표시
- PayCompleted : 결제 완료 후 총 금액을 보여주고, 상품 목록으로 이동할 수 있는 버튼 제공공
